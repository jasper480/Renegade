Unit Converter

Fabric weight across the three units we quote in. Edit any field and the other two update instantly.

Fabric weight

oz/yd² ↔ g/m² (GSM) ↔ oz/ft²

Tip: our product names carry the GSM. PPI-370 is 370 g/m², which is about 10.92 oz/yd².
Assumptions & methodology
  • 1 oz/yd² = 33.906 g/m² (28.3495 g per ounce, 0.836127 m² per square yard).
  • 1 yd² = 9 ft², so oz/ft² = oz/yd² ÷ 9.
  • g/m² is shown as a whole number, ounce values to two decimals, so round trips may drift slightly.

Tool 02

Pricing Converter

Enter the roll width, then type a price into any box. Every other price recalculates. Works in any direction, so you can start from a square foot target and read back the linear yard price.

Price conversions

Full-width pricing. A linear yard is 36 inches long at the full roll width you enter.

Enter a width to start converting.

Assumptions & methodology
  • Price is treated as full-width: the linear price already covers the entire roll width entered.
  • Linear yard area = width × 36 in. Linear meter area = width × 39.3701 in.
  • 1 yd² = 1296 in², 1 ft² = 144 in², 1 m² = 1550.0031 in².
  • All prices shown to four decimals. Nothing is stored or sent anywhere.

Assumptions & methodology

Fabric weight = GSM × roll area, converted to pounds. Width converts at 0.0254 m/in, length at 0.9144 m/yd, weight at 2.20462 lb/kg.

Core weight is estimated as a paperboard tube. A 3 in core is 3.0 in inside diameter with a 0.25 in wall (3.5 in outside). A 6 in core is 6.0 in inside diameter with a 0.5 in wall (7.0 in outside). Core length equals the roll width.

Roll diameter = √(core OD² + 4 × thickness × length ÷ π), wound outward from the core. Thickness comes from the selected product, defaulting to 22 mil when unknown.

Industrial winding cores typically run 0.70 to 0.90 g/cm³. 0.80 is a middle estimate. Adjust if you have a measured core weight, or use Manual core to enter the weight directly.

Assumptions & methodology
  • Shipping weight = sum of all roll weights (fabric plus cores) plus the pallet tare.
  • Rolls are assumed laid flat. The bottom layer holds the "rolls across" count, then layers stack upward. Layers = total rolls ÷ rolls across, rounded up.
  • Length = the longest roll width or the pallet length, whichever is greater.
  • Width = the pallet width, or rolls across × largest roll diameter if that is wider.
  • Height = pallet deck height + layers × largest roll diameter. The largest diameter is used so the estimate stays conservative across mixed rolls.
  • Tare weights: Standard 50 lb, Long 100 lb, XL 150 lb, or your manual entry.
  • For pyramid or nested loads, tick the box and enter the measured height.

Load settings

The cap is the loaded container (cargo plus the container itself). The buffer comes off that, then the container tare is subtracted to get the cargo budget. So with the defaults, the budget is 44,000 minus 5%, minus each container's tare.
